# 실제 분석 프롬프트 예시

## 기본 분석 프롬프트

```
다음 리그오브레전드 ARAM 게임 데이터를 분석해주세요:

[JSON 데이터 첨부]

분석해주실 내용:
1. 각 팀의 승리/패배 요인
2. MVP와 최저 성과 플레이어 선정 및 근거
3. 챔피언별 성과 평가
4. 팀 조합 분석
5. 개선점 및 교훈

구체적인 수치와 함께 분석해주세요.
```

## 특정 영역 집중 분석 프롬프트들

### 1. 개인 성과 분석
```
이 게임에서 [특정 플레이어/챔피언]의 성과를 다음 관점에서 분석해주세요:
- KDA 효율성 및 포지셔닝
- 딜량 대비 골드 효율성
- 팀 기여도 (어시스트, CC, 힐 등)
- 아이템 빌드의 적절성
- 같은 챔피언 평균 대비 성과
- 구체적인 개선 제안
```

### 2. 팀 전략 분석
```
두 팀의 전략적 차이점을 분석해주세요:
- 팀 조합의 강점/약점
- 오브젝트 컨트롤 능력
- 팀파이트 패턴
- 게임 템포 조절
- 승부 결정 요인
- 전략적 실수와 기회 손실
```

### 3. 메타/빌드 분석
```
이 게임의 챔피언별 빌드와 룬을 분석해주세요:
- 각 챔피언의 아이템 빌드 효율성
- 룬 선택의 적절성
- 게임 상황에 맞는 빌드였는지
- 더 나은 대안 빌드 제안
- 상대 팀 조합 고려한 빌드 평가
```

## 비교 분석 프롬프트

### 같은 챔피언 비교
```
이 게임에서 [챔피언명] 플레이어들을 비교 분석해주세요:
- 성과 지표 비교 (KDA, 딜량, 골드 등)
- 플레이 스타일 차이점
- 빌드 선택의 차이와 효과
- 팀 기여도 비교
- 각자의 강점과 약점
```

### 포지션별 비교
```
각 포지션별 플레이어 성과를 비교해주세요:
- 딜러진 비교 (딜량, 생존력, 캐리력)
- 탱커/서포터 비교 (탱킹, 이니시, 보호)
- 포지션별 기대 역할 수행도
- 팀 내 밸런스 평가
```

## 교육적 분석 프롬프트

```
이 게임을 통해 배울 수 있는 교훈들을 정리해주세요:

초보자를 위한 교훈:
- 기본적인 게임 이해도 향상 포인트
- 챔피언별 역할 이해
- 아이템 빌드 기초

중급자를 위한 교훈:
- 팀플레이 개선점
- 상황 판단력 향상
- 효율적인 골드 사용법

고급자를 위한 교훈:
- 미세한 최적화 포인트
- 고급 전략 이해
- 메타 적응력

각 레벨별로 구체적인 예시와 함께 설명해주세요.
```

---

이런 프롬프트들을 Bedrock에 JSON 데이터와 함께 입력하면 상당히 깊이 있는 분석을 받을 수 있을 것 같습니다!
